# Omega

![Logo](https://i.imgur.com/bWpwGI2.png)

My first Discord bot using Discord.js

## Installation

*You'll need Node.js*

[1] Add a file named `.env` with :

```
TOKEN=YOUR_BOT_TOKEN
```

[2] In a command prompt, use `npm install` then `node index`
